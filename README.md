If you want to easily archive you dash cam video and photos, this may be useful to you.

I'm using my *Maxto M3* helmet camera when I ride an electric unicycle (EUC) which is mostly just for fun. 
I save all of my ride videos on an extra hard drive. I don't do much with the video but maybe some day I 
will want to. I developed a strategy to make the backup process easy.

1) Connect Maxto M3 to my computer. It usually shows up as drive E:
2) Run a batch file program. It looks at the memory card and tallies how many files I recorded per day.
3) I tell the program how many days to go back and it attempts to copy all of the files from those days
   to my D: drive. It puts the files from each day in a folder named for that day. It skips files from
   days I've already copied from.
5) Later I can add a description to the folders to help me remember where I was riding. As long as I
   keep the date at the beginning of the folder name it won't copy those files again when I run the
   program later.
